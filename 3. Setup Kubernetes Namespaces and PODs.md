# Setup Kubernetes Namespaces and PODs

## Step 1: Create namespace YAML

**`dev-namespace.yaml`**

```yaml
apiVersion: v1
kind: Namespace
metadata:
  name: dev
```

Apply Namespace: 

```bash
kubectl apply -f dev-namespace.yaml
```

## Step 2: Create pod YAML

**`dev-nginx-pod.yaml`**

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: dev-nginx-pod
  namespace: dev
  labels:
    app: nginx
spec:
  containers:
  - name: nginx-container
    image: nginx:latest
```

Apply Pod:

```bash
kubectl apply -f dev-nginx-pod.yaml
```

## Step 3: Verification

Check namespace:

```bash
kubectl get ns
```

Check pod:

```bash
kubectl get pods -n dev
```
